# Liftoo Frontend

Flutter mobile app for customers and assistants.

## Run

```bash
flutter pub get
flutter run
```

From project root:

```powershell
.\start-frontend.ps1
```

## Android emulator API URL

```bash
flutter run --dart-define=API_BASE_URL=http://10.0.2.2:3000 --dart-define=SOCKET_URL=http://10.0.2.2:3000
```
